# Log Search Tool - 部署说明

## 🚀 快速开始

### Windows 打包版本使用

1. **下载** `deployment.zip` 文件并解压
2. **双击** `start.bat` 启动应用  
3. **访问** http://127.0.0.1:9000
4. **检查状态** 双击 `status.bat`
5. **停止** 应用时双击 `stop.bat`

### 文件说明

```
deployment/
├── log-search-api.exe     # 主程序
├── start.bat             # 启动脚本 (端口9000)
├── stop.bat              # 停止脚本
├── status.bat            # 状态检查脚本
└── README.md             # 本说明文档
```

## ⚙️ 配置

### 端口配置
编辑 `start.bat` 文件，修改这一行：
```bat
set FLASK_PORT=9000
```

### 其他配置
支持通过环境变量配置：

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `FLASK_HOST` | `127.0.0.1` | 监听地址 |
| `FLASK_PORT` | `9000` | 监听端口 |
| `LOG_LEVEL` | `INFO` | 日志级别 |

## 📝 日志

- 应用日志：`logs/app_9000.log`
- 实时查看：`tail -f logs/app_9000.log`

## 🔧 故障排除

### 中文乱码问题
所有批处理脚本已经设置UTF-8编码，如果仍有乱码：
1. 右键点击命令提示符 → 属性 → 字体 → 选择支持中文的字体
2. 或在命令行输入 `chcp 65001` 切换到UTF-8编码

### 端口被占用
1. 修改 `start.bat` 中的 `FLASK_PORT`
2. 或者停止占用端口的程序

### 程序无法启动
1. 检查 `logs/app_9000.log` 日志文件
2. 确保防火墙允许程序运行
3. 以管理员身份运行

### 访问权限
如果需要从其他机器访问，修改 `start.bat`：
```bat
set FLASK_HOST=0.0.0.0
```
